# AI 课题项目管理平台 PRD v1.0

## 1. 目标用户
科研院所项目经理

## 2. 核心用户故事
- 查看项目全景（基本信息、里程碑、任务看板），快速掌握进度。
- 系统 AI 语义分析任务，识别需要开会的任务，提醒并预填会议表单。
- 上传会议纪要后，DeepSeek 自动提取待办任务并分配给成员，形成闭环。

## 3. 功能范围（Must Have）
- 项目详情页：顶部信息卡 + Tab（概览/里程碑/任务看板）
- 任务看板：三列拖拽（待办/进行中/已完成）
- 会议智能提醒：AI 语义判断，预填 Dialog，一键创建会议
- 会议纪要 AI 解析：上传 .txt/.md → DeepSeek 解析 → 可编辑表格 → 确认下发

## 4. Out of Scope
用户登录、真实邮件集成、甘特图

## 5. 技术栈
Next.js 14 + TypeScript + Tailwind + Prisma + SQLite + DeepSeek API
